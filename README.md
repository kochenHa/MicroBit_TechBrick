# MicroBit_TechBrick

**Ein DIY-OpenSource System für Robotik mit Technik-Klemmbausteinen auf der Basis des BBC micro:bit**

---

## Projekt Überblick

Das Ziel ist es ein kompatibles System zu haben mit verschiedenen Verschiedenen Aktoren und Sensoren die in der Microsoft MakeCode Umgebung nahtlos integriert sind und programiert werden können. Die Hardware ist voll kompatibel mit Technik Klemmbausteinen und wird über USB Typ-C Kabel mit einem Kontrol-Brick verbunden. Der Kontrol-Brick beinhaltet das Microbit sowie eine Spannungsversorgung und die Anschlüsse um weiter Module anzuschließen.

Die Hardware kann selber gebaut werden mit einem 3D-Drucker, kommerziell erhältlichen Bauteilen und einer kleinen portion geschick. mehr dazu [hier](link)

---
## Repository Überblick
Das Repository ist in zwei Bereiche aufgeteilt einen für die Maker die das Projekt für sich selber bauen wollen und einen für Entwickler die daran weiter Arbeiten wollen. 

### Maker
In diesem Bereich sind die Druckteile als .STL verfügbar und Anleitungen zum zusammenbau und installation der Software zu finden sowie tipps zur Inbetriebname und Code Beispiele für die einzelnen Module.  

### Developer
In diesem Bereich sind die original CAD-Daten mit einigen (Lizenztechnischen-Ausnahmen) sowie weitere Erklärungen zu der Struktur.

## Software

Zum Programmieren wird die [Microsoft MakeCode for Micro:Bit](https://makecode.microbit.org/) verwendet und die [Extension](https://github.com/la003/hkemt6prmi) benötigt.
