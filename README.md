# MicroBit_TechBrick

**Ein DIY-OpenSource System für Robotik mit Technik-Klemmbausteinen auf der Basis des BBC micro:bit**

---

## Projekt Überblick

Das Ziel ist es ein kompatibles System, mit verschiedenen Aktoren und Sensoren, die in der Microsoft MakeCode Umgebung nahtlos integriert sind und dort programmiert werden können zu entwickeln. Die Hardware ist voll kompatibel mit Technik Klemmbausteinen und wird untereinander über USB Typ-C Kabel mit einem Control-Brick verbunden. Der Control-Brick beinhaltet den Microbit sowie eine Spannungsversorgung und die Anschlüsse um weitere Module anzuschließen.

Die Hardware kann mit einem 3D-Drucker, kommerziell erhältlichen Bauteilen und einer kleinen Portion Geschick selber gebaut werden. mehr dazu [hier](link)

---
## Repository Überblick
Das Repository ist in zwei Bereiche aufgeteilt. Einer für die Maker, die das Projekt für sich selber bauen wollen und einer für Entwickler, die es weiter entwickeln wollen. 

### Maker
In diesem Bereich sind die Druckteile als .STL verfügbar. Anleitungen zum Zusammenbau und zur Installation der Software lassen sich hier ebenso finden wie Tipps zur Inbetriebname und Code Beispiele für die einzelnen Module.  

### Developer
In diesem Bereich sind die originalen CAD-Daten (mit einigen Lizenztechnischen-Ausnahmen) sowie weitere Erklärungen zu der Struktur vorhanden.

## Software

Zum Programmieren wird die [Microsoft MakeCode for Micro:Bit](https://makecode.microbit.org/) verwendet und die [Extension](https://github.com/la003/hkemt6prmi) benötigt.
