# MicroBit_TechBrick

**Ein DIY-OpenSource System für Robotik mit Technik-Klemmbausteinen auf der Basis des BBC micro:bit**

---

## Projekt Überblick

Das Ziel ist es ein kompatiebles System zu haben mit verschiedenen Verschiedenen Aktoren und Sensoren die in der Microsoft MakeCode Umgebung nahtlos programiert werden können. Die Hardware ist voll Kompatiebel mit Technik Klemmbausteinen und wird über USB Typ-C Kabel mit einem Kontrol-Brick verbunden. Der Kontrol-Brick beinhaltet das Microbit sowie eine Spannungsversorgung und die Anschlüsse um weiter module anzuschließen.

Die Hardware kann selber gebaut werden mit einem 3D-Drucker und komerziel erhältlichen Bauteilen und einer kleinen portion geschick. mehr dazu [hier](link)

---

## Software

Zum Programmieren wird die [Microsoft MakeCode for Micro:Bit](https://makecode.microbit.org/) verwendet und die [Extension](https://github.com/la003/hkemt6prmi) benötigt.
